💼 Financial KPI Monitoring & Profitability Dashboard
📊 Project Overview

The Financial KPI Monitoring & Profitability Dashboard is a Power BI project designed to analyze and monitor key financial performance indicators such as revenue, cost, profit, budget variance, and profitability trends across multiple business dimensions.

This dashboard provides a real-world simulation of how organizations track financial health and make data-driven decisions using business intelligence tools.

🎯 Objective

To build an interactive financial analytics dashboard that helps in:

Monitoring revenue and expenses
Tracking profit and margin performance
Comparing budget vs actual performance
Analyzing regional, product, and channel-level profitability
Supporting strategic business decision-making
🛠 Tools Used
Power BI
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
📁 Dataset Description

The dataset includes financial transactions and business performance data with the following columns:

Date, Month, Quarter, Year
Region, Department, Product Category, Product Name
Revenue, Cost of Goods Sold, Operating Expenses
Gross Profit, Net Profit, Profit Margin %
Sales Volume
Budgeted Revenue, Actual Revenue, Budget Variance
Customer Segment, Sales Channel
🔄 Power Query Steps
Removed null and duplicate values
Changed data types (currency, date, percentage)
Created time-based columns (Month, Quarter, Year)
Renamed columns for clarity
Formatted financial fields
Created calculated fields for analysis
📐 DAX Measures Used
Total Revenue
Total Cost
Gross Profit
Net Profit
Profit Margin %
Budgeted Revenue
Actual Revenue
Budget Variance
Revenue Growth %
Cost Ratio %
📊 Dashboard Features
KPI Cards (Revenue, Profit, Margin, Budget Variance)
Revenue by Region (Column Chart)
Profit by Product Category (Bar Chart)
Monthly Trend Analysis (Line Chart)
Sales Channel Distribution (Donut Chart)
Revenue to Profit Breakdown (Waterfall Chart)
Detailed Matrix Table (Region-wise performance)
Interactive slicers (Region, Product, Channel, Month)
📌 Key Insights
Identified top-performing regions contributing highest revenue
Highlighted most profitable product categories
Compared actual performance against budget targets
Analyzed profit margin trends over time
Evaluated performance of different sales channels
